# Novel Analyzer (热门小说分析师)

这是一个专为网文爱好者和创作者设计的 AI Skill，能够深入分析各大平台的热门小说，提炼流行趋势，并为你的创作提供源源不断的灵感。

## ✨ 核心功能

*   **趋势洞察**：一键获取各大主流平台（起点、晋江、番茄等）的月度热门小说榜单。
*   **深度解构**：不仅是简单的罗列，还会分析这些爆款作品的核心卖点、流派设定（如“凡人流”、“无限流”）。
*   **灵感生成**：基于当前市场风向，为你生成定制化的长篇与短篇小说创作构思（包含标题、核心梗、简要大纲）。

## 📚 使用场景

*   **找书荒**：不知道看什么？让 AI 推荐当下最火的作品。
*   **写手必备**：不知道写什么？分析市场热点，避免闭门造车。
*   **选题策划**：快速验证你的创意是否符合当前读者的口味。

## 🚀 快速开始

### 1. 安装 Skill

```bash
npx skills add <your-username>/novel-analyzer
```

### 2. 使用方法

在支持 Skill 的 AI 助手（如 Trae, Cursor 等）中，直接询问：

> "现在的热门小说有哪些？"
> "最近流行什么类型的小说？"
> "我想写一本爽文，给我几个当下最火的创意点子"
